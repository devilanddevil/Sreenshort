# Idle Time Tracker - Installation Instructions

To run this tool on another computer, follow these steps:

## Step 1: Install Python
1.  Download Python from https://www.python.org/downloads/
2.  Run the installer.
3.  **IMPORTANT**: Check the box "Add Python to PATH" before clicking Install.

## Step 2: Install Tesseract OCR
1.  Download Tesseract from: https://github.com/UB-Mannheim/tesseract/wiki
2.  Install it.
3.  By default, the app looks for it at `C:\Program Files\Tesseract-OCR\tesseract.exe`.
    *   If you install it somewhere else, you will need to open `app.py` and change the path.

## Step 3: Run the App
1.  Open the folder containing these files.
2.  Double-click `run_app.bat`.
    *   This will automatically install the necessary libraries and start the server.
3.  The app should open in your browser (or go to http://127.0.0.1:5000).

## Note on First Login
The first account you create will automatically become the **Admin**.
